# Como Contribuir

Obrigado pelo interesse em contribuir com este projeto!

## Passos

1. Faça um fork do repositório.
2. Crie uma branch descritiva: `git checkout -b feat/minha-contribuicao`.
3. Faça suas alterações e commit com mensagem clara.
4. Abra um Pull Request descrevendo o que foi alterado e por quê.

## Boas práticas

- Mantenha os commits pequenos e focados.
- Escreva mensagens de commit no imperativo.
- Certifique-se de que o código está funcionando antes de abrir o PR.
